API REFERENCE

Use INCO.HOST programmatically in your bots, scripts & apps

POST /api/upload
Upload a single file. Returns URL and metadata.
# cURL curl -X POST https://your-domain.com/api/upload \ -F "file=@image.png" # Response { "success": true, "url": "https://your-domain.com/f/uuid.png", "id": "uuid", "type": "image", "size": 204800, "deleteKey": "delete-uuid" }
POST /api/upload/multi
Upload multiple files at once (max 10). Returns array of URLs.
curl -X POST https://your-domain.com/api/upload/multi \ -F "files=@img1.jpg" \ -F "files=@video.mp4"
GET /api/file/:id
Get metadata for a specific file by ID.
curl https://your-domain.com/api/file/uuid
DELETE /api/delete/:id?key=deleteKey
Delete a file using its ID and delete key.
curl -X DELETE https://your-domain.com/api/delete/uuid?key=deleteKey
GET /api/stats
Get platform statistics and file counts.
